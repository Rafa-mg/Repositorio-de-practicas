Atributos y sus datos
  • el atributo name se utiliza para darle la identificacion a la informacion cuando la estemos pasando a la base de datos esto es particularmente util cuando vamos a cargar un formulario a una base.
  • atucomplete: nos permite otorgarle a nuestro formulario la capacidad de usar los datos que yahan sido cargados previamente en el navegador, si estamos usando un navegador que soporte el autocompletado claro
  • required: nos permite indicarle al usuario que debe llenar uno a demas datos requeridos antes de hacer click en submit. Tambien podemos usar las propiedades valid e invalid en css para darle otros estilos a estas interacciones
