Tipos de inputs

otras consideraciones atener en cuenta son:

  • no anidar anchor tags u otros elementos que puedan causar complicaciones a los usuarios para activar los inputs
  • No anidar headings
  • Si un formulario o una seccion del formulario necesita un titulo, usa el elemento legend dentro de una etiqueta fieldset.
  • un input de tipo boton con un atributo de value valido, no necesita una label asociada. De hecho puede interferir con las tecnologias de asistencia. Lo mismo aplica para los elementos de boton
